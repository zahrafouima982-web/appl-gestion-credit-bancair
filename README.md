# 🏦 Application de Gestion des Crédits Bancaires

Ce projet est une application en **langage C** développée pour automatiser les opérations bancaires (clients, crédits, paiements).

## 👤 Réalisé par :
* **Zahra Fouima**
* **Encadré par** : Mr. BAHRI, Abdelkhalak
* **Établissement** : ENSA d'Al Hoceima (2025/2026)

## 📂 Organisation du Projet
Le projet utilise une architecture modulaire conforme au rapport :
* `client.c/h` : Gestion de la base client.
* `credit.c/h` : Cœur du moteur financier.
* `paiement.c/h` : Suivi des remboursements.
* `style.h` : Interface console ergonomique.
