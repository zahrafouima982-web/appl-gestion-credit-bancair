# 🏦 Application de Gestion des Crédits Bancaires

Ce projet est une application en **langage C** développée pour automatiser les opérations bancaires liées aux clients, aux crédits et aux paiements.

## 👤 Réalisé par :
* **Zahra Fouima**
* **Encadré par** : Mr. BAHRI, Abdelkhalak
* **Établissement** : ENSA d'Al Hoceima (2025/2026)

## 📂 Organisation du Projet
Le projet utilise une architecture modulaire composée des fichiers suivants :
* [span_0](start_span)`client.c/h` : Gestion de la base client.[span_0](end_span)
* [span_1](start_span)`credit.c/h` : Cœur du moteur financier.[span_1](end_span)
* [span_2](start_span)`paiement.c/h` : Suivi des remboursements.[span_2](end_span)
* [span_3](start_span)`style.h` : Interface console stylisée.[span_3](end_span)
